# Tracking my ISP connection with a Visualization Dashboard 

A prototype for a Python-based visualization dashboard using the dash-package.
Best performance using Python 3.9

## To install the required packages run the following command in the project's directory:
```
$ pip install -r requirements
```

The dashboard currently runs on some dummy data.

## To run the dashboard use the following commands:
```
$ python app.py
```

You should get a message saying `Dash is running on http://127.0.0.1:8050/`

In your browser go to this address, which can also be accessed by going to `localhost:8050`
